# PhaseNet

Implementation of a phase-aware anomaly detection pipeline for industrial sensor data.

## Main Files

- `train.py` trains the model.
- `inference.py` runs anomaly scoring.
- `evaluate.py` computes evaluation metrics.
- `dataset.py` loads the prepared dataset.
- `model/` contains the network modules.
- `preprocessing/` contains data preparation utilities.

## Note

This repository is configured to keep source code in Git and exclude large local assets such as datasets, PDFs, archives, notebooks, and cache files.
